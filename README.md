# Travel Recommendation Website

A responsive travel recommendation web application that helps users discover popular beaches, temples, and countries around the world.

## Features
- Responsive Navigation Bar
- Home Page
- About Us Page
- Contact Us Page
- Search Functionality
- Beach Recommendations
- Temple Recommendations
- Country Recommendations
- Mobile-Friendly Design
- GitHub Pages Deployment.

## Technologies Used
- HTML5
- CSS3
- JavaScript
